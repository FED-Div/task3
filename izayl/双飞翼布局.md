### 双飞翼布局

与圣杯布局一样，双飞翼布局也是用来解决三列布局的问题(左右两列定宽，中间列宽度自适应)。

下面简析其实现的原理

**HTML**

```html
<div id="header"></div>

  <div class="center">
    <div class="center-content">center</div>
  </div>
  <div class="left">left</div>
  <div class="right">right</div>

<div id="footer"></div>
```

注意这里的中间列用了容器包裹，双飞翼布局的关键就在这。

```css

body{min-width: 700px;}
.left,
.center,
.right{
  float: left;
  min-height: 130px;
}

.left{
  margin-left: -100%; //与圣杯布局一致
  width: 200px;
  background-color: #ccc;
}
.right{
  margin-left: -220px; //圣杯布局中使用右负边距加相对定位实现
  width: 220px;
  background-color: #ccc;
}

.center{
  width: 100%;
}
.center-content{
  margin: 0 200px 0 200px;
  min-height: 130px;
  background-color: blue;
}
  
```

:halloween:看看实现的详细流程：

1. 浮动三列
2. 使用负边距把左列移至正确位置 margin-right: -100%;
3. 使用负边距把右列移至正确位置 margin-left: -220px;
4. 使用子元素实现中间列的位置
   1. 中间列容器设置宽度 100%
   2. 子元素设置左右margin以实现位置 margin: 0 220px 0 200px;
